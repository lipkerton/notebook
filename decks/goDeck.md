Как сделать переменную, которая будет хранить строку `"Привет!"`? #flashcard 
```Go
var var_name string = "Привет!"
```

Что будет если не указывать тип переменной при ее создании? #flashcard 
Тогда компилятор автоматически определит тип по значению переменной:
```Go
var var_name = "Привет!"
```

Можно ли в Go, как в C, сначала объявить переменную, а потом задать ей значение? #flashcard 
Можно.

В чем различие между типом `int` и `uint`? #flashcard 
`uint` может содержать только положительные числа.

Можно ли сложить число типа `int` с числом типа `uint`? #flashcard 
Нет, компилятор вернет ошибку.

Как в Go выполнить множественное присваивание значений нескольким переменным? #flashcard 
```Go
var number1, number2, number3 int = 1, 2, 3
```

Какие два типа переменных есть в Golang? #flashcard 
Глобальные и локальные. Локальными переменными называются все те переменные, которые объявлены внутри фигурных скобок `{`. Это означает, что их можно использовать только внутри этих же скобок.

Какой есть специальный способ объявления локальной переменной? #flashcard 
```Go
func my_func() {
	phrase string := "Привет!"
}
```

Как константа отличается от переменной? #flashcard 
Константа объявляется словом `const`, а не `var`. Значение константы нельзя изменить после ее объявления.

Можно ли производить математические операции над переменными разных типов? #flashcard 
Нет, нельзя.

Какая функция и какого модуля позволяет делать строки с форматированием (вроде `f-string` в Python)? #flashcard 
Модуль `fmt`, функция `Printf()`:
```Go
package main
import "fmt"
func main() {
	test_var := "Петр!"
	fmt.Printf("Привет, %s", test_var)
}
```

Как можно в функции `fmt.Printf` напечатать знак `%` (например, я хочу, чтобы в консоль вывелось число `25%` при том, что число `25` у меня сохранено в переменной)? #flashcard 
```Go
func main() {
	a := 25
	fmt.Printf("Проценты: %d%%", a)
}
```

Что выражает знак `%v` в контексте вывода форматированной строки с помощью функции `fmt.Printf()`? #flashcard 
С его помощью можно напечатать и строки, и числа. Как я понял - это универсальный символ, которым можно обозначать сразу несколько типов данных:
```Go
func main() {
	test1 := 1
	test2 := "Привет!"
	test3 := 2.35
	fmt.Printf("%v\n", test1)
	fmt.Printf("%v\n", test2)
	fmt.Printf("%v\n", test3)
	fmt.Printf("%v %v", test1, test2)
}
```

Что выражает знак `%d` в контексте вывода форматированной строки с помощью функции `fmt.Printf()`? #flashcard 
С его помощью можно выразить числовые типы данных - `int, float, uint`:
```Go
func main() {
	test1 := 1
	test2 := 1.5
	fmt.Printf("%d\n", test1)
	fmt.Printf("%d\n", test2)
}
```

Каким образом в консоль можно выводить ошибки? #flashcard 
С помощью функции `fmt.Errorf()` - она возвращает строку, которую можно вывести в консоль с помощью `fmt.Println()`:
```Go
func main() {
	var name string = "Peter"
	var user_id int = 5
	err := fmt.Errorf("User %q (id %d) not found!", name, user_id)
	fmt.Println(err.Error())
}
```

Как мне вывести в консоль строку, но оставить кавычки? #flashcard 
Это можно сделать с помощью `%q`:
```Go
var name string = "peter!"
fmt.Println("%q", name)
```
