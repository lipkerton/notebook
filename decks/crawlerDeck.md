Как начать проект в scrapy? #flashcard 
Для начала нужно создать и активировать виртуальное окружение + скачать само расширение scrapy:
```Bash
python -m venv .venv
source .venv/<bin|Scripts>/activate
pip install scrapy
```
Теперь можно начать проект:
```Bash
scrapy startproject <имя_проекта>
```
