Что такое указатель? #flashcard 
Указатель это штука, которая указывает на конкретную область в памяти. Например:
```C
int a = 1;
int *p;
p = &a;
```
Здесь я объявляю указатель `int *p`, который должен указывать на адрес памяти, где хранится значение типа `int`. Потом я получаю адрес памяти значения от переменной `a` - `&a` и записываю его в указатель.

Как я могу перемещаться по массиву в C? #flashcard 
С помощью арифметики указателей:
```C
int values[5] = {10, 20, 30, 40, 50};  // массив из пяти элементов.
printf("%p\n", values);  // указатель на первый элемент массива.
printf("%p\n", values + 1);  // указатель на второй элемент массива.
printf("%d\n", *(values + 1));  // второе число в массиве.
```
Или с помощью индексов (что то же самое):
```C
int values[5] = {10, 20, 30, 40, 50};
printf("%d\n", values[1]);  // второе число в массиве.
```

Что делает такая запись `a |= 1`? #flashcard 
Оператор `|=` это побитовое ИЛИ с присваиванием.
Например:
```C
int a = 0;
a |= 1;  // 00000000 | 00000001 = 00000001 = 1;
a |= 2;  // 00000001 | 00000010 = 00000011 = 3;
a |= 1;  // 00000011 | 00000001 = 00000011 = 3;
```
Побитовое ИЛИ сравнивает каждый бит двух чисел по правилу - если один из двух битов равен `1`, то результат равен `1`. Если оба бита равны `0`, то результат равен `0`.
