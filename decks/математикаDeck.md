Как обозначается абсолютная величина числа? #flashcard

Правило сложения отрицательных чисел? #flashcard 
Чтобы сложить два отрицательных числа нужно сложить их абсолютные величины и перед полученной суммой поставить знак минус.
```
(-3) + (-7) = |-3| + |-7| = 3 + 7 = -10
```
<!--ID: 1749898853469-->


Правило сложения отрицательного и положительного чисел? #flashcard 
Чтобы сложить два числа, из которых одно положительное, а другое отрицательное и которые имеют разные абсолютные величины, надо из большей абсолютной величины вычесть меньшую и перед полученным результатом поставить знак того из слагаемых у которого абсолютная величина больше.
```
25 + (-10) = 25 - 10 = 15
```

Какие есть основные правила умножения в алгебре? #flashcard 
1) произведение двух отрицательных чисел равно произведению их абсолютных величин.
2) произведение двух чисел, из которых одно положительное, а второе отрицательное число, равно отрицательному числу, абсолютная величина которого равна произведению абсолютных величин сомножителей.
3) произведение двух чисел равно нулю, если хотя бы одно из них равно нулю.

Какие есть основные свойства умножения? #flashcard 
Три основных закона:
1) коммутативный - от перемены мест множителей произведение не меняется.
2) ассоциативный - порядок выполнения умножения не влияет на произведение.
3) дистрибутивный -  произведение суммы чисел на число равно сумме произведений слагаемых на это число.

Как работает дистрибутивный закон на примере? #flashcard 
Дистрибутивный закон на практике означает, что умножение суммы чисел на число означает сумму умножения каждого слагаемого на это число:
```
(10 + 1) * 2 = 10 * 2 + 1 * 2 = 22
```

Можно ли устанавливать правило вычитания? #flashcard 
Правило вычитания вытекает из правил сложения, поэтому правило вычитания нельзя устанавливать по определению. Другими словами любое вычитание - это сумма:
```
10 - 8 = 2 => 10 = 2 + 8
10 + (-8) = 2
```

Можно ли устанавливать правило деления? #flashcard 
Правило деления вытекает из правила умножения, поэтому правило деления нельзя устанавливать, т.к. оно вытекает из правила умножения и определения действия деления. Деление - это действие, которое обратно умножению, поэтому любое деление есть лишь обратное умножение:
```
10 : 2 = 5 => 10 = 5 * 2
10 * (1/2) = 5
```
