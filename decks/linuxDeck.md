7) `/usr/bin/zsh` - оболочка, которая будет запускаться при заходе пользователя в систему.

Расшифруй запись: `-rw-r--r-- 1 peterpetukhov peterpetukhov 12 May 17 11:51 hello` #flashcard 
1) `-rw-r--r--` - это режимы доступа к файлу. Первый символ - указывает на то, что файл является файлом, три символа после (`rw-`) указывают на то, что может делать с файлом владелец, еще три следующих определяют, что с файлом может делать группа, которая им владеет (`r--`) и последние три - что с файлом могут делать все остальные (`r--`).
2) `1` - номер файла в папке.
3) `peterpetukhov` - имя владельца.
4) `peterpetukhov` - имя группы.
5) `12 May 17 11:51` - дата и время создания файла.
6) `hello` - имя файла.

Как разархивировать архив в Linux? #flashcard 
С помощью утилиты `tar` и ее дополнительных параметров:
1) `-x` - (extract) извлечение.
2) `-z` - (gzip) это движок распаковки (может быть разным).
3) `-v` - (verbose) будет писать в терминал отчет по распаковке.
4) `-f` - (file) после этого пишется имя файла, который мы будем распаковывать.
```Bash
tar -xzvf <имя архива>
```

Как заархивировать файлы в Linux? #flashcard 
С помощью утилиты `tar` и ее дополнительных параметров:
1) `-c` - (create) создание.
2) `-f` - (file) после этого аргумента пишется имя файла, который мы будем распаковывать.
```Bash
tar -cf <имя будущего архива> <файл\папка>
```

Как добавить пользователя в `sudoers`? #flashcard 
Нужно обновить файл `/etc/sudoers`, добавив строку в формате:
```Bash
<имя пользователя которому дают права> <хост который может пользоваться правами>=(<пользователь от имени которого выполняются команды>:<группа от имени которой выполняются команды>) <набор доступных команд>
```
Пример:
```Bash

```
