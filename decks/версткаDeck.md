Какой тип перехода используется по умолчанию, когда пользователь переходит по ссылке? И как его изменить? #flashcard 
По умолчанию новая страница в браузере заменяет собой ту, где была нажата ссылка. Это поведение можно изменить, если прописать атрибут `target` в теге `<a>`. Так же можно дополнительно прописать поведение по умолчанию в том же теге. Значение `_blank` обяжет клиент открыть новую вкладку, а значение `_self` откроет новую страницу на текущей. Пример:
```HTML

```

Что такое селектор? #flashcard 
Селектор в CSS - это имя определенного тега (/тегов) или путь до определенного тега (/тегов). Селекторы CSS часто используются, чтобы определить для какого тега или совокупности тегов установить тот или иной стиль. А еще они могут быть использованы для поиска тегов.
```HTML
<style>
  селектор по имени тега
  li {
    color: red;
  }
  селектор по двум именам тега
  он выделяет все теги a, которые вложены в тег li
  li a {
    color: red;
  }
</style>
```
<!--ID: 1748606404821-->


Какие существуют виды селекторов? #flashcard 
У селекторов существует свой собственный синтаксис, самые главные команды:
1) `*` - выбор всех тегов.
2) `li *` - выбор всех тегов, которые вложены в `li`.
3) `.class_name` - выбор всех тегов с классом `class_name`.
4) `li.class_name` - выбор всех тегов `li` с классом `class_name`.
5) `li .class_name` - выбор всех тегов с классом `class_name`, которые вложены в тег `li`.

Что такое DOM? #flashcard 
Это формат, в котором браузер интерпретирует структуру HTML. Такой формат браузер создает в памяти при анализе HTML документа. Формально он очень похож на вывод команды `tree` - это просто дерево, которое состоит из тегов, где каждый тег может иметь какие-то вложенные теги и т.д. Тег, вложенный в другой тег называется "потомок", а его "верхний" тег - "родитель". У родителя может быть много потомков, но у потомка может быть только один родитель.
<!--ID: 1748876290872-->


Как устроен приоритет селекторов? #flashcard 
Приоритет селекторов нужен в случаях, когда у нас существует, например, несколько конфликтующих стилей для одного элемента на странице. Тег будет выбран в соответствии со своей специфичностью - чем меньше элементов потенциально может включить в себя тег, тем более специфичен (и приоритетен) он будет. Самый приоритетный - индивидуальный стиль тега через атрибут `style`. Наименее приоритетный - селектор по имени тега.

Что такое наследование в CSS? #flashcard 
Наследование - один из основных принципов CSS (именно поэтому стили называются каскадными). Он означает, что стили могут наследоваться от родительских элементов во все дочерние элементы. Существуют CSS-правила, которые наследуются и которые не наследуются. Различить их можно по принципу: те правила, которые можно указать и для родительского тега и для тега потомка, - являются не наследуемыми (например,  `background` - можно назначить и для родителя, и для потомка отдельно).
