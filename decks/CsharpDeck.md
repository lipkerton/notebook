Что такое `.NET`? #flashcard

Что такое CLR в `.NET`? #flashcard 
Common Language Runtime - общеязыковая среда выполнения. Она берет промежуточный скомпилированный код `C#` и выполняет его (компилирует его в нормальный машинный код и передает на выполнение в процессор). Она же следит за сборкой мусора в проекте.

Что такое "точка входа"? #flashcard 
Точкой входа в любой программе на `C#` традиционно считается метод `Main`. Компилятор автоматически ищет этот метод среди всех файлов программы и начинает его компиляцию сразу же, как найдет. Метод `Main`, в отличие от других методов, является приоритетным для среды выполнения, поэтому его можно описывать и как `private`, и как `public` - среда выполнения его все равно увидит + никаким другим методам он по стандарту не нужен, т.к. просто выполняет функцию точки входа в программу.

В чем различие между структурой и классом? #flashcard 
Экземпляры класса хранятся в *HEAP memory*, а референсы (переменные) на экземпляры класса хранятся в *STACK memory*; переменные *ссылаются* на объекты класса в памяти, что позволяет нескольким переменным ссылаться на один экземпляр класса. Структура же хранит все свои экземпляры в *STACK memory* и поэтому на ее объекты не существует ссылок.

Что такое *выражение, оператор, операнд* в С#? #flashcard 
Выражение - это участок кода, который возвращает какое-то значение. Выражение всегда должно что-то делать. Выражение состоит из операторов, а операторы из операндов.
При этом какие-то выражения могут операторами не являться. Оператор - это участок кода, который заканчивается точкой с запятой, поэтому, например, выражение `2 + 2` не может быть оператором.

Что такое `#if #elif #else #endif`? #flashcard 
Это конструкция, которая используется вместе с символами компиляции, чтобы изменить поведение компилятора при компиляции программы.
```C#
#if DEBUG
	Console.WriteLine("Привет, это конфигурация DEBUG!");
#endif
```
Строка будет напечатана в консоли, если выставлен символ компиляции `DEBUG` (он выставлен по-умолчанию в конфигурации `Debug`).

Какие есть три базовых символа компиляции, которые определяет .NET по умолчанию? #flashcard 
.NET определяет сразу три конфигурации для запуска проекта - `Release`, `Debug`, `Trace`. Все конфигурации устанавливают для себя по символу - `DEBUG`, `RELEASE`, `TRACE`. Конфигурация `Trace` - это система трассировки .NET, поэтому проект, который запущен с конфигурацией `Trace`, читает весь код проекта, вне зависимости от того отмечен ли он `DEBUG` или `RELEASE`.

Как я могу написать такой код, который будет выполнен только в конфигурации `Debug`? #flashcard 
Такой код можно написать с помощью `#if #elif #else #endif` или с помощью конструкции `[System.Diagnostics.Conditional(<имя маркера>)]`:
```C#
#if DEBUG
	<мой код>
#endif
// ИЛИ
[System.Diagnostics.Conditional("DEBUG")]
static void my_method()
{
	<мой код>
}
```
