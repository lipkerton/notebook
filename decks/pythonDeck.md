Зачем нужен venv? #flashcard 
Чтобы держать все нужные для проекта пакеты в одном месте внутри проекта. Что такое ранг массива и как его увидеть? Как узнать общее количество элементов в массиве? #flashcard 
Ранг массива - это подсчет уровней вложенности. У массива, который внутри содержит два массива ранг будет равен двум, т.к. мы имеем внешний массив и внутри него еще массивы. Если бы внутренние массивы содержали бы еще массивы, то это был бы уже третий ранг. Узнать ранг массива можно с помощью атрибута `ndim`. Узнать общее количество элементов в массиве можно с помощью атрибута `size`. Ранг и количество элементов можно получить из атрибута `shape`.
```Python
import numpy
m = numpy.array([[1, 2, 3], [4, 5, 6]])
print(m.ndim)  # 2
print(m.size)  # 6
print(m.shape)  # (2, 3)
```


Что такое ООП и зачем оно нужно? #flashcard

Что такое наследование (принципы ООП)?  #flashcard 
Наследование - это еще один основной принцип. В сущности он говорит о том, что систему, которая состоит из множества классов можно спроектировать через наследование маленьких классов от классов более общих. Основная фишка в том, что класс-потомок наследует все характеристики и методы родительского класса, но добавляет еще ряд своих особенных характеристик. Что такое `__pycache__`? #flashcard 
Это папка, которая содержит скомпилированные файлы Python. Она создается после первого запуска программы на Python: программы с расширением `.py` компилируются в байт-код и сохраняются в `__pycache__`. Следует понимать, что сохраняются кэши только импортируемых модулей, а не модулей верхнего уровня. Например, кэш файла `main.py` (если через него была запущена программа) сохранен не будет, а вот если `main.py` импортирует какой-то другой модуль, то этот уже модуль будет сохранен в виде кэша.
