Зачем нужен venv? #flashcard 
Чтобы держать все нужные для проекта пакеты в одном месте внутри проекта. Например, это упрощает транспортировку проекта с компа на комп или в GitHub, т.к. из папки виртуального окружения можно вытащить все зависимости и их версии, чтобы сохранить их в специальном файле. Еще venv по умолчанию необходим, например, на macOS, который просто не позволяет устанавливать файлы с помощью `pip` без виртуального окружения.


Что происходит при активации виртуального окружения? #flashcard 
Скрипт активации виртуального окружения изменяет переменную `PATH`. Он ставит путь до папки с виртуальным окружением перед путями к стандартным папкам Python. Делает он это для того, чтобы Python сначала искал пакеты в активированном виртуальном окружении, а потом уже в своих стандартных папках.

Что делает функция `vars()`? #flashcard 
Она выводит на экран значения локальных переменных + значения параметров функции.
```Python
>>> def func(*args, **kwargs):
...     print(vars())
...
>>> func(1, 2, 3)
{'args': (1, 2, 3), 'kwargs': {}}
```

Назови основные команды, которые есть в стандартном отладчике Python. #flashcard
`s`(step) - шаг вперед к следующей строчке кода.
`n`(next) - прыгаем через текущую фукнцию.
`c`(continue) - продолжаем работу программы до точки останова, ошибки или завершения программы.
`b <номер_строки>` - ставим точку останова на номере строки.
`l <номер_строки(опционально)` - показываем кусочек кода и все точки останова на нем.

Какие уровни сообщений есть в `logging`? #flashcard 
Их пять:
1) `DEBUG`- уровень, который отражает всю информацию, которая происходит в программе.
2) `INFO` - отражает менее детализированную информацию. Базово означает, что программа работает как нужно.
3) `WARNING` - что-то пошло не по плану. Какая-то функция отработала не так, как нужно.
4) `ERROR` - случилась ошибка. Программа не может выполнить какую-то функцию.
5) `CRITICAL` - случилась критическая ошибка. Программа не может продолжать свою работу.

Как включить запись в файл для всех логов уровня `INFO` и ниже? #flashcard
Нужно прописать параметры для функции `basicConfig()`.
```Python
import logging
logging.basicConfig(
	level='INFO',
	filename='<имя файла>'
)
```
