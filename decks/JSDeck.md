В чем различие между двойным равенством и тройным равенством в JS? #flashcard 
Разница в том, что двойное равенство проверяет операторы на равенство между собой, а тройное равенство проверяет их идентичность. В частности, при сравнении нуля и `false` через двойное равенство мы получим `true`, т.к., как и в Python, пустые последовательности равны `false`, а при сравнении того же через тройное равенство мы получим `false`, потому что для получения `true` в этом случае нам нужно было бы сравнивать `false` и `false`:
```JavaScript
0 == false;  // => true
0 === false;  // => false
false === false;  // => true
```

Как я могу спросить есть ли нужный мне атрибут в объекте JS? #flashcard 
С помощью оператора `?`. Если есть какой-то атрибут, в существовании которого мы сомневаемся, то можно вызвать его с знаком `?`. В этом случае при отсутствии атрибута JS просто ничего не вернет, а при наличии вернет сам атрибут. Пример:
```JavaScript
x = {a: "Привет!"};
x?.b  // спрашиваем есть ли атрибут b в объекте.
x?.a  // => "Привет!" спрашиваем есть ли атрибут a в объекте.
x.a?.b  // спрашиваем есть ли атрибут b у атрибута a.
```

Как сделать функцию в JS? #flashcard 
С помощью ключевого слова `function`. Функции могут принимать аргументы. Аргументы идут после называния функции в круглых скобках. После аргументов открывается фигурная скобка и пишется тело функции:
```JavaScript
function plus1 (x) {
	return x + 1;
};
```
Вызывается функция стандартным способом - с помощью круглых скобок:
```JavaScript
plus1(10);
```
