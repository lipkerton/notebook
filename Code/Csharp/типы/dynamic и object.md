# dynamic
тип данных который является отдельной от всех других типов сущностью.
когда C# только начинал развиваться он был настроен специально для работы с ОС Windows, а в ней уже использовались некоторые динамические структуры. поэтому для нормального управления динамическими структурами был придуман динамический тип `dynamic`:
```C#
dynamic c = 3;
c = "mother";
```
в сущности он повторяет динамическое поведение о объектов из динамических языков. например, компилятор не будет проверять будут ли успешны выполняемые динамическим кодом операции.
однако следует знать, что динамическая часть языка C# считается устаревшей и почти не развивалась с момента дебюта в 2010 году.
# object
`object` это базовый класс для всех типов в C#. переменная `object` может ссылаться на значение любого типа, производного от `object`.
