в C# есть такие же кортежи, как и в Python.
чтобы объявить кортеж нужно:
```C#
(int a, int b) c = (5, 7);
```
чтобы вызвать каждый элемент кортежа, можно обратиться к ним по именам, которые указаны при объявлении:
```C#
Console.WriteLine(c.a);
Console.WriteLine(c.b);
```
важно понимать, что в кортеж типа `(int, int)` не получится записать три значения типа `int` и точно так же не получится записать значение `char` или `string`. зато, если попытаться записать числовые типы (по правилам неявного преобразования в C#, конечно), то все получится отлично.
так же можно использовать `var` для быстрого объявления кортежа:
```C#
var c = (5, 7);
```
в этом случае доступ к элементам можно получить по атрибуту `Item<номер_элемента>`:
```C#
Console.WriteLine(c.Item1);
Console.WriteLine(c.Item2);
```

---
в C# есть аналог распаковки кортежа. он называется *деконструкция*. это когда мы берем переменные по количеству элементов кортежа и кладем значения кортежа в переменные:
```C#
var c = (5, 7);
(int x, int y) = c;
```
