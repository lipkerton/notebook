# while
цикл `while` вычисляет выражение и ожидает получить значение типа `bool`. если он получает `true`, то произойдет вычисление следующего блока кода. когда блок кода завершится `while` снова вычислит выражение и если оно будет `true`, то снова выполнит блок кода. так будет, пока вычисление выражения не вернет `false`.
```C#
int counter = 0;
while (counter < result)
{
	Console.WriteLine(counter);
	counter += 1;
}
```
так же как и в других языках программирования есть:
- `break` - немедленно завершает текущую итерацию и цикл.
- `continue` - немедленно завершает текущую итерацию и начинает следующую (перемещается обратно к проверке условия цикла).
# do
иногда требуется, чтобы перед проверкой условия в цикле `while` была выполнена хотя бы одна операция:
```C#
char k;
do
{
	Console.WriteLine("Press x to exit");
	k = Console.ReadKey().KeyChar;
}
while (k != 'x');
```
это, кстати, интересная программа - в ней я считываю ввод с консоли (точнее одну букву ввода) и записываю букву в переменную `k`. если буква была корректной, то `while` завершит работу, а если нет, то цикл начнется заново.
# for
цикл, который в книге называется "унаследованным от C":
```C#
for (инициализация переменной; условие завершения цикла; итератор) {...}
```
в самом общем случае пример был бы таким:
```C#
for (int i = 1; i < 10; i++) {...}
```
здесь я объявляю локальную переменную цикла, условие завершения цикла и ту операцию, которая будет выполняться каждую итерацию цикла.
```C#
for (int i = 1; i < 10; ++i)
{
	Console.WriteLine(i);
}
```

# foreach
альтернатива стандатному циклу `for`, которая похожа на конструкцию в Python:
```Python
for i in <последовательность>:
```
в C#:
```C#
int[] d = {1, 2, 3};
foreach (int item in d) 
{
	Console.WriteLine(item)
}
```
для того, чтобы `foreach` перебрала коллекцию, коллекция должна содержать `GetEnumerator` метод.
важно помнить, что `foreach` не способен модифицировать коллекцию, потому что не все коллекции, которые он способен перебрать поддерживают модификацию.
