ссылка https://leetcode.com/problems/add-two-numbers/
даны два связанных списка:
```Python
# Definition for singly-linked list.
class ListNode:
     def __init__(self, val=0, next=None):
         self.val = val
         self.next = next
```
нужно их сложить, чтобы получился третий связанный список, который является суммой двух первых.
