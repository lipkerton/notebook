Пандас - это штука, которая является чем-то вроде универсального чтеца для разных форматов хранения данных. С ее помощью можно прочитать, например, csv-файлы и т.д. Но не все так просто. Pandas очень полюбили создатели курса “Аналитика данных” в ВШЭ. Почему? Потому что с его помощью можно читать и анализировать большие наборы данных. Это норм, если нам нужно просто прочитать набор, а затем сделать какие-то выводы на основе информации в нем; в таком сценарии нам не нужно вносить изменения в условную базу данных или представлять, что к нашему формату хранения данных будут подключаться пользователи и т.д.

> [!important] Номинальные переменные - это переменные, которые записаны словами в таблице.
> 
> Константные переменные - это переменные, которые записаны цифрами.

В пандасе есть два главных объекта `DataFrame` и `Series` . Первый - это стандартная таблица, второй - это просто список элементов (\вектор\массив).

# read_csv, head, tail, iloc

Допустим у нас есть большой csv, где есть 9000 строчек. Нам нужно прочитать его и каким-то образом с ним поработать. Я не хочу делать это через менеджер контекста, потому что тогда у меня будет куча необработанных строк. Я хочу получить на выходе таблицу, как это было бы, если бы я хотел загрузить мой csv документ в эксель. В пандас такой фокус можно провернуть одной командой.

## read_csv

```Python
import pandas


data = pandas.read_csv('path')
print(data)
```

Что за магия происходит в переменной `data`? В переменной `data` храниться объект класса `DataFrame` - этот класс предлагает формат демонстрации данных. Pandas автоматически переводит формат csv в необходимую ему форму, чтобы показать буквально таблицу с данными на экране; таблица и появится в результате работы этого скрипта.

Функцию `read_csv()` можно применить даже если существует необходимость считать csv-файл по ссылке из интернета.

```Python
import pandas


data = pandas.read_csv(
	'https://raw.githubusercontent.com/rogovich/Data/master/data/titanic/train.csv'
)
print(data)
```

## head\tail

Это две команды из линукса. Они просто выводят заданное количество строк “сверху” файла и “снизу” файла. В данном случае у нас таблица, поэтому будут выведены строки из ее начала и из ее конца.

```Python
import pandas


data = pandas.read_csv('path')
print(data.head(5))
print(data.tail(5))
```

Вернется точно так же объект класса `DataFrame`.

## iloc

По сути это константа, для которой можно указать индекс, как для списка, а она по этому индексу вернет конкретную строчку данных из таблицы со всеми ее подробностями - значения колонок и названия колонок.

```Python
import pandas


data = pandas.read_csv('name_file.csv')
print(data.iloc[255])  # вернет 255 строку данных, но счет-то с нуля идет.
```

Я немного не понял отличие, но существует еще функция `loc()` - она делает то же самое, что и `iloc()` , т.е. находит строку по номеру и может дополнительно взять из нее колонку или колонки.

```Python
import pandas


data = pandas.read_csv('name_file.csv')
print(data.loc[255, 'column_name'])
```

# shape, info, columns

Это константы для того, чтобы смотреть различную информацию конкретно о табличке. С ними все конкретно просто: `shape` - возвращает размерность таблицы (сколько на сколько), `columns` - возвращает список (массив) колонок таблицы. C `info` правда все чуть-чуть сложнее - на самом деле это не константа, а функция, которая буквально самостоятельно печатает всю информацию о табличке в терминал.

```Python
import pandas


data = pandas.read_csv('path')
print(data.shape)
print(data.columns)
data.info()
```

Если набрать `info()` то в результатах будут:

- колонка `Non-Null Count` - эта колонка считает для каждого столбца количество не-пустых ячеек.
- колонка `Dtype` - эта колонка показывает типы данных, которые хранятся в каждой колонке таблички.

# всякие логические комбинации

Данные из таблички можно выбирать по признакам. Сами признаки можно конструировать в логических комбинациях. Это проще чем звучит:

- Символ `&` - это логическое “И”. Вернет правду, если только справа и слева от него правда.
- Символ `|` - это логическое “ИЛИ”. Вернет правду, если только справа и слева от него правда.
- Символ `~` - это логическое отрицание. Меняет местами ложь и правду.

Для примера давайте выберем все дни, когда были праздники:

```Python
print(data['Holiday'] == 'Holiday')
```

В результате вернуться две колонки - в первой будут индексы, а во второй будет проставлено `True`. Почему так? Потому что мы попросили напечатать не саму колонку ‘Holiday’ - мы попросили проверить для каких строчек утверждение `data[’Holiday’] == ‘Holiday’` является верным. В результате пандас прошелся по всем строчкам проверил каждую и напечатал `True` рядом с номерами строчек, где были праздники.

Теперь, как серьезно сделать так, чтобы нам показались именно строчки из таблицы, в которых помечено, что был праздник?

```Python
print(data[data['Holiday'] == 'Holiday'])
```

Вот это уже вернет именно строчки, потому что так в пандас выглядит **поиск по условию**. Мы ищем только те колонки, которые удовлетворяют утверждению.

А так мы будем искать колонки по двум признакам: или был выходной, или был нерабочий день.

```Python
print(data[(data['Holiday'] == 'Holiday') | (data['Functioning Day'] == 'No')]
```

Здесь использовано логическое “ИЛИ”. Следует сразу заметить, что оба участника операции с логическим “ИЛИ” вынесены в скобки и еще следует сразу заметить, что использовать стандартные для Python логические операторы нельзя - будет ошибка.

Теперь используем отрицание, чтобы найти дни, когда праздников не было.

```Python
print(data[~(data['Holiday'] == 'Holiday')]
```

Если найдется дата для которой утверждение `data[’Holiday’] == ‘Holiday’` верно, то логическое отрицание перевернет это на ложь; наоборот, если условие окажется ложным, то логическое отрицание перевернет его на истину - наше условие сработает с точностью наоборот и так покажет все даты без праздников.

# поиск по ячейке, столбцу, строке

Осуществляется командой `iloc` и специальными записями вроде индексации.

```Python
import pandas


data = pandas.read_csv('path')
print(data['column_name'])  # поиск всех данных по имени колонки.
print(data[['column_name_1', 'column_name_2']])  # поиск сразу по двум колонкам.
print(data.iloc[1212]['column_name'])  # ищем строчку с индексом и вытаскиваем колонку.
```

## поиск с условием - isin

Функция `isin()` получает значение, либо список значений и фильтрует определенную колонку. В результате возвращается массив (\вектор) значений `true\false` . Например, мы захотели узнать в каких строках у нас сезон “зима”, а в каких нет.

```Python
import pandas


data = pandas.read_csv('path')
print(data['Seasons'].isin(['Winter']))
```

Точно так же можно проверить, например, строки на наличие в них либо зимы, либо лета.

```Python
print(data['Seasons'].isin(['Winter', 'Summer']))
```

Всю конструкцию с условием можно передать в условия поиска и получить на выходе полноценные строки, где в колонке погоды указаны либо зима, либо лето.

```Python
print(
	data[
		data['Seasons'].isin(
			['Winter', 'Summer']
		)
	]
)
```

## поиск пустых ячеек - isna

Функция `isna()` ищет пустые значения в колонках. Все. Возвращает так же булевый массив, по которому можно осуществлять поиск.

```Python
import pandas


data = pandas.read_csv('path')
print(data['Temperature'].isna())
```

Проверяем все ячейки на наличие в них значений или на отсутствие.

Теперь можно провернуть поиск с условием и найти все строки, где температура оказалась пустой.

```Python
print(
	data[
		data['Temperature'].isna()
	]
)
```

# сортировка и удаление дубликатов

`sort_values()` - выполняет сортировку. У функции два аргумента - `by` и `ascending` . Первый указывает колонки для выполнения сортировки, а второй определяет сортировку по убыванию или по возрастанию.

Сортировка всей таблицы выглядит так:

```Python
import pandas


data = pandas.read_csv('data.csv')
print(data.sort_values(by='Temperature'))  # сортировка по одной колонке.
print(data.sort_values(by=['Temperature', 'Hour'])  # сортировка по нескольким.
print(data.sort_values(by=['Temperature', 'Hour'], ascending=[True, False]))
```

В последнем случае мы сортируем сразу по двум колонкам и для каждой выставляем свой режим. Для первой колонки “по-возрастанию”, а для второй - “по-убыванию”. В аргументе `by` значения указываются по приоритету сортировки, т.е. первая колонка будет отсортирована в приоритете, а вторая будет сортировать только те значения, которые совпадают в первой.

Сортировка конкретной колонки:

```Python
import pandas


data = pandas.read_csv('data.csv')
print(data['Temperature'].sort_values())
print(data['Temperature'].sort_values(ascending=False))
```

`drop_duplicates()` - выполняет удаление дубликатов.

```Python
import pandas


data = pandas.read_csv('data.csv')
print(data.drop_duplicates('Temperature'))
```

# частотная таблица

Так в курсе названа функция `value_counts()` . Если совсем просто объяснять: эта функция анализирует колонку и возвращает список с подсчетом “сколько раз в колонке указано каждое уникальное значение”. У нее есть еще не обязательный аргумент `normalize` - его фишка в том, что он переводит те значения, которые посчитала функция в формат десятичной дроби - это похоже на подсчет по процентам, только проценты будут выражены в десятичном формате.

```Python
import pandas


data = pandas.read_csv('data.csv')
print(data['имя колонки'].value_counts())
print(data['имя колонки'].value_counts(normalize=True))
```

Этот метод возвращает вектор типа Series - это значит, что он выглядит как массив в Python или как список. Фишка такого вектора в том, что мы можем достать из него отдельно индексы и отдельно значения с помощью атрибутов `index` и `values` .

```Python
import pandas


data = pandas.read_csv('data.csv')
print(data['имя колонки'].value_counts().index)
print(data['имя колонки'].value_counts().values)
```

В результате вернуться массивы - первый будет массив индексов (принадлежит к классу `Index` - это класс Pandas), а второй массив значений (этот массив - это `array` из NumPy).

# добавить колонки

добавить колонки в таблицу Pandas можно буквально точно так же как и в словарь Python. Задаем название ключа и присваиваем ему значение.

```Python
import pandas


data = pandas.read_csv('data.csv')
data['new'] = 0
print(data['new'])
```

При таком коде Pandas создаст новую колонку ‘new’ и для каждой строки этой колонки проставит значения 0. Этим поведением можно воспользоваться, чтобы для каждой колонки задать новые значения.

```Python
import pandas


data = pandas.read_csv('data.csv')
data['new'] = data['number_1'] + data['number_2']
print(data['new'])
```

В новую колонку будут записаны суммы значений из двух старых колонок. Здесь можно по-разному извращаться, например, можно использовать лямбду, чтобы трансформировать значения для новой колонки. Чтобы так преобразовывать значения понадобиться функция `apply()` .

```Python
import pandas


data = pandas.read_csv('data.csv')
data['new'] = data['number_1'].apply(lambda x: x if x > 5 else 0)
print(data['new'])
```

# группировки

## агрегатные функции

В pandas есть все стандартные функции группировок:

- `max()` - максимальное значение.
- `min()` - минимальное значение.
- `sum()` - сумма.
- `count()` - подсчет значений.

И есть еще особенные:

- `mean()` - функция нахождения среднего значения по колонке.
- `median()` - медиарное значение (???), не понимаю что это.
- `unique()` - функция нахождения уникальных значений колонки.
- `nunique()` - функция которая считает количество уникальных значений.

Применяются они довольно просто - нужно взять колонку (или несколько колонок) и после применить функцию.

```Python
import pandas


data = pandas.read_csv('data.csv')
print(data['Column'].max())
print(data['Column'].min())
print(data['Column'].mean())
print(data['Column'].count())
print(data['Column'].unique())
print(data['Column'].nunique())
print(data['Column'].sum())
```

## groupby

Функция `groupby()` применяется к DataFrame и создает объект, к которому можно применить какую-нибудь функцию группировки. Например, мы можем сначала сгруппировать всех пассажиров по порту из которого они отправились в путешествие.

```Python
import pandas


data = pandas.read_csv(
	'https://raw.githubusercontent.com/rogovich/Data/master/data/titanic/train.csv'
)
print(data.groupby(['Embarked']))
```

В результате этой операции не получится никаких данных, т.к. получится только сгруппированный объект. Из этого объекта можно извлечь данные по получившимся группам. Например, найти число пассажиров, которые совершили посадку через этот порт. Для этого нужно посчитать количество пассажиров для каждой группы.

```Python
data.groupby('Embarked')['PassengerId'].count()
```

Точно так же можно сгруппировать сначала по порту отправления, а внутри групп по порту сгруппировать еще данные по полу пассажира, а потом все посчитать.

```Python
data.groupby(['Embarked', 'Sex'])['PassengerId'].count()
```

В сущности я могу применить любую функцию группировки к конкретным колонкам внутри конкретной группы. Чтобы это показать я найду внутри тех же групп среднюю цену билета по каждой группе и средний возраст.

```Python
data.groupby(['Embarked', 'Sex'])[['Fare', 'Age']].mean()
```

Можно даже округлить получившиеся значения, например, до двух знаков после запятой, с помощью функции `round()`.

```Python
data.groupby(['Embarked', 'Sex'])[['Fare', 'Age']].mean().round(2)
```

Функция `agg()` позволяет применить сразу несколько функций к выбранным колонкам по группам. на выходе получится табличка, где строки будут представлять конкретную группу, а колонки будут представлять результаты работы функций по группе.

```Python
data.groupby(['Embarked', 'Sex'])[['Fare', 'Age']].agg('count', 'mean').round(2)
```

Еще функция `agg()` позволяет применять функции группировки отдельно для каждой колонки. Чтобы это провернуть нужно передать ей словарь, где имена ключей - это имена колонок, а значения - это имена функций или списки с именами функций.

```Python
data.groupby(['Embarked', 'Sex']).agg({'Fare': 'sum', 'Age': 'mean', 'min', 'max'})
```
