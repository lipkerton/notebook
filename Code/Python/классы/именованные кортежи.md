Именованные кортежи - это структура данных, которая импортируется из модуля `collections` и может быть использована для создания компактных неизменяемых классов.
Идея именованных кортежей появилась из двух проблем, которые есть у обычных кортежей:
1) когда мы создаем обычный кортеж, то размещаем в нем элементы в каком-то порядке. Этот порядок всегда непросто запомнить, потому что элементов может быть много. 
2) доступ к элементам кортежа осуществляется только по индексам (или в цикле), что может ухудшить читаемость кода.
Обе проблемы в сущности можно свести к тому, что нам просто нужна структура кода, которая может быстро создать неизменяемый класс, объявить имена его атрибутов.
```Python
from collections import namedtuple


car_1 = namedtuple('Car' , ['color', 'speed'])
car_2 = namedtuple('Car', 'color speed')
```
Это два равноправных способа создать именованный кортеж. Первый аргумент - имя, которое будет присвоено классу, а второй аргумент (либо строка, либо список) - это набор атрибутов для этого класса.
Теперь экземпляр класса можно создать так:
```Python
car_1('red', 100)
car_2('blue', 200)
```
После присваивания атрибуты у данного класса нельзя будет поменять. Однако я могу вытащить их по индексу:
```Python
car_1[0]  # red
car_1[1]  # 100
```
А еще можно "распаковать" такой объект, как обычный кортеж:
```Python
color, speed = car_1
```
Объект, который создается через `namedtuple` имеет все свойства нормального класса, например, мы можем унаследовать о этого объекта какой-нибудь класс и даже определить его методы.
```Python
from collections import namedtuple

car = namedtuple('Car', 'color speed')
def NewCar(car):
	def hexcolor(self):
		uf self.color == 'red':
			return '#ff0000'
		return '#000000'
```
