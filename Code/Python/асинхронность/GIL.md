С некоторых пор в Python была встроена штука, которая называется **GIL**.. Это расшифровывается как **Global Interpreter Lock**.
C этой штукой связан термин **мьютекс**. **Мьютексы** - это искусственные ограничения, которые накладываются работу с ресурсами, чтобы другие компоненты не вторгались в процесс работы.
**GIL** - это встроенный в Python мьютекс, который дает возможность выполнять python-код только одному потоку за раз.
