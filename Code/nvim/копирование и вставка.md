Я копировал в `wsl` и столкнулся с проблемой - не копируется в буфер обмена. В интернете посоветовали решение - использовать команду `"+y`. 
Однако если запускать `nvim` в `wsl` стандартным образом, то копирование будет выдавать ошибку - `nvim` не сможет получить доступ к некоторым нужным ему исполняемым файлам. 
Чтобы исправить ошибку я просто запустил `nvim` через `sudo`.
Чтобы вставить из буфера обмена нужно использовать комбинацию `"+p`
