`git merge --squash <hash>` - мержит текущую ветку с веткой из хэша, но все коммиты входящей ветки складываются в один коммит.
`git reset --hard <hash>` - **ОСТОРОЖНО!** Эта команда переделывает весь текущий локальный репозиторий в тот, что хранится по хэшу. Эта штука оказалась полезной, когда я делал ветку от другой ветки (а не от `main` ветки) и при MR новой стал видеть все изменения обеих веток в `changes`. Не прикольно видеть те изменения, которые ты не вносил в этой ветке. А эта команда помогает начать историю изменений с чистого листа.
