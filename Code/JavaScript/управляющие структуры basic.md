В JS есть большинство стандартных для других языков управляющих структур - условные конструкции, цикл for, цикл while и проч.
В данный момент я знаю, что их можно использовать внутри функций. Например, я хочу реализовать функцию, которая будет проверять число, и отдавать на выход абсолютную версию числа:
```JavaScript
function abs (x) {
	if (x >= 0) return x;  // начало конструкции if
	else return -x;  // конец конструкции if
};
abs(-10);  // => 10
abs(-10) === 10;  // => true
```
Управляющая конструкция `if` устроена так же как и прочие подобные в других языках - есть обязательный блок `if`, после которого идет код. В моем примере сразу после `if` идет `return` - это допустимо, если код внутри `if` занимает всего одну строку. То же самое происходит с `else`. `else` - это необязательный блок кода.
Теперь я хочу реализовать функцию, которая будет суммировать числа в последовательности. Сделаю я это с помощью цикла `for`:
```JavaScript
function sum (array) {
	let i, result = 0;
	for (i of array) {
		result += i;
	}
	return result;
};
sum([1, 2, 3]);  // => 6
```
Точно так же как и в примере с `if` конструкцию `for` можно уместить в одну строку - необязательно помещать код внутри `for` в фигурные скобки, если код состоит из одной строки:
```JavaScript
for (i of array) result += i;
```
Внутри конструкции `for` используется синтаксис, который означает, что в переменную `i` будут поочередно записаны все элементы последовательности `array`.
Теперь о реализации цикла `while`. Все стандартно: нужно ключевое слово `while` и какое-то условие, которое находится сразу после `while`. Например, я хочу написать функцию, которая будет расчитывать факториал числа - для этого мне нужно перемножить все числа от единицы до самого числа включительно.
```JavaScript
function factorial (number) {
	let start_number = 1;
	while (number > start_number) {
		start_number *= n;
		n--;
	};
	return start_number;
};
factorial(4);  // => 24
```
То же самое можно реализовать и с циклом `for`, только в этом случае его нужно будет реализовать иначе, чем в примере с суммой чисел в последовательности:
```JavaScript
function factorial (number) {
	let i, start_number = 1;
	for (i = 2; i <= number; i++) {
		start_number *= i;
	};
	return start_number;
};
```
Запись в цикле означает следующее:
1) `i = 2` - стартовое значение `i` внутри цикла.
2) `i <= number` - условие, согласно которому цикл закончится.
3) `i++` - операция, которая будет производится на каждом шаге цикла.
