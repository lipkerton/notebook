JS - это динамический, нетипизирвоанный, высокоуровневый, интерпретируемый язык программирования. Изначально на нем можно было программировать в браузерах, потому что основная среда его размещения - это именно клиентское приложение. Однако потом для JS придумали собственную среду разработки `Node`, которая позволяет JS осуществлять операции не только в рамках клиентского приложения, но и со всей операционной системой.
