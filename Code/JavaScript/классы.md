В JS существует разновидность ООП. Она не совсем похожа на то, что принято считать под ООП в "классических" языках программирования.
Создать класс в JS  можно с помощью ключевого слова `class`. Далее нужно указать имя класса и поставить фигурные скобки - в них будет описан класс.
В JS существует конструктор класса - его можно описать с помощью ключевого слова `constructor` и передать ему все нужные аргументы, если аргументы необходимы:
```JavaScript
class Point {
	constructor(x, y) {
		this.x = x;
		this.y = y;
	};
};
```
В такой же манере, как и конструктор, мы можем описать все дальнейшие методы класса. Например, я пропишу метод, который вычисляет расстояние от начала координат до точки:
```JavaScript
class Point {
	constructor(x, y) {
		this.x = x;
		this.y = y;
	};
	distance() {
		return Math.sqrt(
			this.x * this.x + this.y * this.y
		);
	};
};
```
Теперь я могу создавать объекты класса `Point` с помощью ключевого слова `new`:
```JavaScript
let p = new Point(1, 1);
```
Теперь по такой же схеме я создам еще один простейший класс:
```JavaScript
> class Test {
... constructor(name) {
... this.name = name
... }}
undefined
> let t = new Test('Peter')
undefined
> a.push
a.push

> a.push(t)
3
> a
[ [Function: x], [Function: y], Test { name: 'Peter' } ]
```
