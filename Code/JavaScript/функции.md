## база
Функции в JS - это такие же примерно штуки, как и в Python. То есть это какой-то именованный параметризированный блок кода, который может вернуть значение или просто произвести вычисление.
Точно так же как в Python функции будут реагировать и совершать действия только когда произошел их вызов.
Базовый пример функции:
```JavaScript
function plus1 (x) {
	return x + 1;
}
plus1(5);  // => 6
```
Функции можно присваивать переменным:
```JavaScript
let square = function (x) {
	return x * x
};  // знак ";" здесь нужен, чтобы обозначить конец присваивания.
square(5);  // => 25
```
Функции можно передавать в качестве значений другим функциям:
```JavaScript
square(plus1(6))  // => 49
```
Еще один пример функции:
```JavaScript
function sum_two (x, y) {
	return x + y;
};
let x;
x = sum_two(5, 10);
x  // => 15
```
## сокращенная функция
Для функций так же существует сокращенный синтаксис через символ ` =>` и такие функции обычно используются, чтобы быть аргументами для других функций. 
```JavaScript
const plus2 = x => x + 2;
plus2(4)  // => 6
```
Такие функции можно передавать на вход и другим функциям:
```JavaScript
const plus2 = x => x + 2;
const square = x => x * x;
square(plus2(2));
```
## методы
### вызываем методы стандартных типов
Как и в Python, для стандартных типов данных в JS существуют собственные методы. Например, для списка есть методы `push` и `reverse`. Первый метод "толкает" новые элементы в массив JS, а второй метод "переворачивает" массив JS.
```JavaScript
let a = [];
a.push(1, 2, 3);
a  // => [ 1, 2, 3 ]
a.reverse()  // => [ 3, 2, 1 ]
```
### определение собственных методов
Методы - это функции, которые присваиваются определенным объектам. Мы можем приcвоить метод любому объекту в JS, если сошлемся на его атрибут.
```JavaScript
let a = {x: 0, y: 1};
a.some_func_1 = function () {
	return 10 + 1;
};
a.some_func_1();  // => 11
```
Но что если я хочу получить доступ именно к элементам объекта `a`, чтобы сделать с их значениями что-то? Тогда можно воспользоваться ключевым словом `this`:
```JavaScript
let a = {x: 0, y: 1};
a.mult = function () {
	return this.x * this.y;
};
a.mult()  // => 0
```
Теперь я хочу построить что-то посерьезнее - функцию, которая вычисляет теорему Пифагора:
```JavaScript
let points = [
	{x: 0, y: 0},
	{x: 1, y: 1}
];
points.sqrt = function () {
	let p1 = this[0];  // берем первый элемент массива
	let p2 = this[1];  // берем второй элемент массива
	let a = p2.x - p1.x;
	let b = p2.y - p1.y;
	return Math.sqrt(a * a + b * b);  // Math.sqrt вычисляет квадратный корень.
};
points.sqrt()  // => 1.4142135623730951
```
Кстати, то же самое можно было бы реализовать и без модуля `Math`:
```JavaScript
return (a * a + b * b) ** 0.5;
```
