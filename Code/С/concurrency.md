третья программа из книжки:
```C
#include <stdio.h>
#include <stdlib.h>
#include "common.h"

volatile int counter = 0;
int loops;

void *worker(void *arg) {
	for (int i = 0; i < loops; i++) {
		counter++;
	}
	return NULL;
}

int main(int argc, char *argv[]) {
	if (argc != 2) {
		fprintf(stderr, "usage: threads <value>\n");
		exit(1);
	}
	loops = atoi(argv[1]);
	pthread_t p1, p2;
	printf("Initial value: %d\n", counter);
	
	Pthread_create(&p1, NULL, worker, NULL);
	Pthread_create(&p2, NULL, worker, NULL);
	Pthread_join(p1, NULL);
	Pthread_join(p2, NULL);
	printf("Final value: %d\n", counter);
	return 0;
}
```
правда в ней используются инструкции, которых у меня нет:
```C
Pthread_create();
Pthread_join();
pthread_t
```
чтобы их получить я добавил в заголовочные файлы `pthread.h` и использовал методы этого файла:
```C
pthread_create();
pthread_join();
pthread_t
```
итак, что тут происходит?
внутри функции `main()` я сначала проверяю с помощью `if` был ли передан аргумент внутрь программы. затем я беру число, которое должно быть числом итераций, и делаю  с ним это:
```C
loops = atoi(argv[1]);
```
`atoi()` - это функция, которая преобразовывает строку к числу. *atoi = ASCII to integer*. в данном случае я преобразую аргумент командной строки к числу (все аргументы командной строки являются строковыми).
сразу после создаются две переменные типа `pthread_t` они должны хранить идентификаторы потока, которые потом создаст функция `pthread_create()`:
```C
pthread_t p1, p2;
pthread_create(&p1, NULL, worker, NULL);
pthread_create(&p2, NULL, worker, NULL);
```
записью `&p1` я обозначаю, что передаю в функцию номер ячейки памяти, куда следует записать идентификатор потока.
после того как потоки созданы нужно дождаться пока они завершаться. это происходит при выполнении функции `pthread_join()`:
```C
pthread_join(p1, NULL);
pthread_join(p2, NULL);
```
что за `NULL` аргументы?
функция `pthread_create()` принимает:
- указатель на область в памяти
- указатель на объект атрибутов потока
- указатель на функцию, которую будет выполнять поток
- указатель на аргументы функции (если их несколько) или конкретный аргумент (если он один)
функция `pthread_join()` принимает:
- переменную, которая хранит поток
- указатель на переменную, куда будет записано значение, возвращаемое потоком
если проставлены `NULL` то будут использованы значения по умолчанию.
запустить многопоточную программу на компиляцию можно с помощью параметра `-pthread`:
```C
gcc -pthread <имя программы>
```
кстати, `printf()` отличается от `fprintf()` тем, что она подразумевает вывод в стандартный поток вывода `stdout` по умолчанию, а `fprintf()` позволяет выбрать поток вручную.
