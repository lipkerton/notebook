это ключевое слово, которое говорит компилятору, что переменная может быть изменена без каких-либо действий со стороны кода, который окружает эту переменную. при этом такая переменная может быть изменена в любое время.
для объявления `volatile` переменной следует:
```C
volatile int foo;
int volatile foo;
```
между этими объявлениями нет разницы.
можно так же сделать `volatile` указатель:
```C
volatile int *foo;
int volatile *foo;
```
и `volatile` указатель на `volatile` переменную:
```C
volatile int *volatile foo;
```
так же есть `volatile` указатели на не-`volatile` переменные:
```C
int *volatile foo;
```
при использовании слова `volatile` на переменной я отключаю для нее разные оптимизации, которые обычно накладывает на переменные. например, он схлопывает дубликаты кода в один код.
например:
```C
int main() {
	int a = 0;
	a |= 1;
	a |= 2;
}
```
компилятор схлопнет две операции в одну - `a |= 3`.
