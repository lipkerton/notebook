| term                                    | description                                                                                                                                                                                                                                                             |
| --------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [[loopback]]                            | Это слово используется для описания принципа работы [localhost](http://localhost) и адреса 127.0.0.1. Базово оно означает возвращение сигнала обратно отправителю. Ну и по принципу работы localhost можно понять, что это по факту обращение компьютера к самому себе. |
| [[chunk]]                               | Кусок информации, например, 1024 байта. Это слово используют, чтобы описать информацию, которая поделена на равные части. Например, если большое сообщение приходит на сервер - его читают по-частям, т.е. делят на чанки.                                              |
| [[privileged ports (well-known ports)]] | Применяется для описания диапазона портов от 0 до 1023.                                                                                                                                                                                                                 |
