Когда я захожу в БД через `psql` система всегда просит пароль:
```Bash
psql -h 127.0.0.1 -U peterpeyukhov -d test
```
Это не очень удобно, если мы говорим о "девелоперской" сборке, т.е. если мы говорим о сборке для тестов, практики и проч.
Эту ситуацию можно изменить, если сделать файл `pgpass` в домашней директории пользователя и записать туда данные для входа в систему:
```Bash
echo "<IP>:<порт>:<имя БД>:<имя пользователя в postgres>:<пароль>" > ~/.pgpass
```
