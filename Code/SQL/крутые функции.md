## coalesce()
`coalesce()` заменяет все значения `NULL` в колонке на что-нибудь еще в результирующей таблице:
```SQL
SELECT coalesce(description, '-')
FROM author;
```
Важно, что значение колонки и заменяемое значение должны быть одного типа, чтобы сформировалась колонка из однотипных значений. Например, у меня колонка `description` - это текстовая колонка, поэтому отсеивая из нее все значения `NULL` из нее я должен заменять их текстовыми значениями, иначе все те ячейки колонки `description`, которые были заполнены текстовыми значениями не будут совпадать по типу с тем, что пытается запихнуть `coalesce()` вместо `NULL`.
## nullif()
`nullif()` ставит `NULL` в ячейку, если первый аргумент функции оказался равен второму. Например, я хочу вместо нулей в табличке иметь значение `NULL`, тогда я напишу такой запрос:
```SQL
SELECT nullif(book_count, 0)
FROM book;
```
