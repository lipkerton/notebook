## Background
- ИИ стал частью системы академического письма.
- Трудности с верификацией работ.
- Структуризация знания с помощью технологии.
- Использование структуризации, как основы для производства знания.
## Problem Statement
**Проблема (тезис)**: теоретическое знание производится с помощью ИИ и от других экземпляров теоретического знания, которые были произведены с помощью ИИ. 
**Цель**: менее формальные подходы к производству знания, чтобы избежать унификации производства знания.
- **Задача 1** - как социальность производит знание?

	- Palermos S.O., Active externalism, virtue reliabilism and scientific knowledge (https://drive.google.com/file/d/1997hRGejDK-JXeYz-oHVQzB6JWbFNNEC/view?usp=sharing).
